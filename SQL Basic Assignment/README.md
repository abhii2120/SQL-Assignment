# SQL Basics Assignment

This folder contains the files related to the **SQL Basics Assignment**.

## Files Included:
- **SQL Basics.sql**  
  This file contains SQL queries based on basic SQL concepts such as SELECT statements, WHERE clause, ORDER BY, DISTINCT, and basic conditions.

- **Answer Explanation (PDF)**  
  This document provides clear explanations for each SQL query, explaining the logic and approach used to solve the questions.

- **Question Paper (PDF)**  
  This file contains the original SQL Basics assignment questions provided by the institute.

## Objective:
The objective of this assignment is to understand and practice the fundamental concepts of SQL and their practical application.

All files are clearly named and organized for easy review and evaluation.
