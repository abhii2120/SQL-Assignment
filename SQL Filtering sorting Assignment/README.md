# SQL Filtering & Sorting Assignment

This folder contains the files related to the **SQL Filtering and Sorting Assignment**.

## Files Included:
- **Filtering_Sorting.sql**  
  This file contains SQL queries based on filtering and sorting operations using clauses such as WHERE, ORDER BY, AND, OR, BETWEEN, LIKE, and IN.

- **Answer Explanation (PDF)**  
  This document provides clear explanations for each SQL query, describing the logic used for filtering and sorting the data.

- **Question Paper (PDF)**  
  This file contains the original SQL assignment questions related to filtering and sorting operations.

## Objective:
The objective of this assignment is to understand how data can be filtered and sorted effectively using SQL queries.

All files are clearly named and organized for easy review and evaluation.
