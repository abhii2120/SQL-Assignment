# SQL Functions Assignment

This folder contains the files related to the **SQL Functions Assignment**.

## Files Included:
- **Functions.sql**  
  This file contains SQL queries using different SQL functions such as string functions, date functions, numeric functions, and aggregate functions.

- **Answer Explanation (PDF)**  
  This document provides step-by-step explanations for each SQL query, explaining the logic and usage of the SQL functions applied.

- **Question Paper (PDF)**  
  This file contains the original assignment questions related to SQL functions.

## Objective:
The objective of this assignment is to understand and apply various SQL functions to manipulate, retrieve, and analyze data effectively.

All files are clearly named and organized for easy review and evaluation.
